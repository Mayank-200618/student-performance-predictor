# 🎓 Student Performance Predictor

This project predicts a student's final score based on study hours, attendance, and previous academic performance using Machine Learning.

---

## 🚀 Features
- Predicts student performance using Linear Regression
- Takes user input for real-time prediction
- Simple and beginner-friendly implementation

---

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

---

## 📊 How It Works
1. Dataset is created with student details
2. Data is split into training and testing sets
3. A Linear Regression model is trained
4. User inputs values
5. Model predicts the final score

---

## ▶️ How to Run

1. Install required libraries:
pip install pandas scikit-learn


2. Run the program:

python project1.py


3. Enter:
- Study hours  
- Attendance  
- Previous score  

4. Get predicted result 🎯

---

## 📌 Example
Input:

Study Hours: 5
Attendance: 70
Previous Score: 60


Output:

Predicted Final Score: 63
